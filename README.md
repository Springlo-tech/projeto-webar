AR Product Viewer - Purificador Philips
Este projeto é um protótipo de Realidade Aumentada (WebAR) que permite aos usuários visualizar um purificador de água da Philips em escala real dentro de seu próprio ambiente, utilizando apenas o navegador do celular.

O objetivo é demonstrar como a Realidade Aumentada pode auxiliar na decisão de compra, permitindo verificar dimensões, design e ajuste estético de um eletrodoméstico em uma cozinha real antes da aquisição.

🚀 Tecnologias Utilizadas
Google <model-viewer>: Componente web para exibição e interação com modelos 3D.

HTML5/CSS3: Estrutura e estilização da interface.

Blender: Utilizado para a otimização e texturização do modelo 3D (formatos .glb).

GitHub Pages: Hospedagem do protótipo funcional.

✨ Funcionalidades
Visualização 3D Interativa: Rotação e zoom do produto no navegador desktop ou mobile.

Realidade Aumentada (AR): Suporte para ARCore (Android) e ARKit (iOS) via Quick Look.


Sem Instalação: Funciona diretamente via link, facilitando o compartilhamento e acesso do cliente.

Para testar: Aponte a câmera do celular para o qrcode imagens> qr.png
